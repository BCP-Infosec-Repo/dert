# Dert

Tool used to enumerate hosts and domains for reconnaissance during a penetration test.

## Installation

Add this line to your application's Gemfile:

    gem 'dert'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dert

## Usage

    require 'dert'
or
    cd bin
    chmod +x ./dert
    ./dert
